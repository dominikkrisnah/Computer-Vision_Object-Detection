# Computer-Vision_Object-Detection

This repository contains Object Detection projects that use various architectures to detect objects in images or videos using Computer Vision and Deep Learning approaches.
This project was created as a practical implementation of an object detection model that can be used for learning, experimentation, and further development.

## 📌 Key Features

- Object detection using various architectures (e.g., YOLO, CNN, etc.)
- Supports inference on images, videos, or webcams
- Simple and easy-to-develop project structure
- Suitable for both basic learning and advanced experimentation in computer vision

## 🧠 Technology Used

- Python
- NumPy
- Scipy
- Scikit Learn
- OpenCV
- Tensorflow
- Ultralitics
- Jupyter Notebook
